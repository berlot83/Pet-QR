Aplicación APK Android, para consumo de respuestas de MicroServicios producidos por aplicación Webapp TrackPetService

La misma está definida para realizar llamadas a pets2018.herokuapp.com

Forma de uso:

Se crea un código QR desde la WebApp y se consume desde esta App.

Utiliza:
Thread,
Zxing,
Gson,
Volley (Ajax android Google),
Gradle,

En este estado sólo falta compilarla en un APK. La misma podría dejar de funcionar si en algún momento se actualiza la dirección de llamadas, hasta que se cambie por la actualizada.
